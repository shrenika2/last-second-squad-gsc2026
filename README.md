# InternMatch AI 🎓🤖

> **Google Solution Challenge 2026** — Team: Last Second Squad

AI-powered campus placement & internship matching platform for students, employers & faculty.

## 🚀 About

InternMatch AI addresses the challenge of inefficient campus recruitment by providing an intelligent, role-based platform that automates student-job matching using AI, streamlines campus drive management for employers, and gives administrators full governance oversight — all in one unified system.

## ✨ Features

- 🎓 **Student Dashboard** — AI match scores, interview prep hub, team formation, explore roles
- 🏢 **Employer & Faculty Dashboard** — Manage campus drives, talent pipeline, question bank
- 🛡️ **Admin Governance Center** — User management, approvals, system health, analytics
- 🤖 **Gemini AI Integration** — Smart resume matching, skill gap analysis, AI assistant
- 🌑 **Premium Dark UI** — Glassmorphism, animated orbs, gradient design system
- 🔐 **Mock Auth** — Role-based login with simulated Google Auth

## 🛠️ Tech Stack

- **Frontend:** React 18, Vite, Tailwind CSS
- **Animations:** Framer Motion
- **Charts:** Recharts
- **AI:** Google Gemini API
- **Icons:** Lucide React

## 🏃 Run Locally

```bash
npm install
npm run dev
```
